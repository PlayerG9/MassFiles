# FileStructure
## file.zip  
```text
+ images
| + image.png
+ files
| + hello_world.tktxt
| + hello_world
| | + hello_world_child.tktxt
+ configs
| + settings.json
```

## file.tktxt (pickle)
```text
(key, value, index)(key, value, index)(key, value, index)
```
Possible keys:
- tagon / tagoff
- mark
- text
- image
